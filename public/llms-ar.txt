# فيصل السويد - معرض أعمال مطور برمجيات

> معرض أعمال احترافي يعرض مشاريع تطوير البرمجيات وتطبيقات الويب وتطبيقات الجوال وحلول الذكاء الاصطناعي.

## نبذة عني

فيصل السويد مطور برمجيات ومهندس تطبيقات متكاملة من المملكة العربية السعودية. متخصص في:

- **تطوير الويب**: React، Next.js، TypeScript، JavaScript، Tailwind CSS
- **تطوير الجوال**: Flutter، Dart، React Native
- **تطوير الخوادم**: Node.js، Python، Flask، Express
- **حلول الذكاء الاصطناعي**: TensorFlow، YOLOv5، تعلم الآلة، رؤية الحاسوب
- **البنية التحتية والسحابة**: AWS، GCP، Azure، Docker، MongoDB، MySQL

## الخدمات

- تطوير تطبيقات ويب مخصصة
- تطوير تطبيقات الجوال (iOS و Android)
- حلول الذكاء الاصطناعي وتعلم الآلة
- تطبيقات الأكشاك التفاعلية
- منصات التلعيب المؤسسي
- حلول التجارة الإلكترونية
- تطوير وتكامل APIs

## المشاريع المميزة

### نظام DroneHunter
نظام كشف وتتبع وتحييد الطائرات المسيرة في الوقت الفعلي باستخدام أجهزة استشعار متقدمة وتعلم الآلة.

### موقع Focus Consultancy
موقع مؤسسي حديث ثنائي اللغة مع React وTypeScript وعناصر ثلاثية الأبعاد.

### منصة MEWA للتلعيب
منصة تلعيب الثقافة المؤسسية تتضمن ألعاب تفاعلية واختبارات وتصور أحداث في الوقت الفعلي.

### مركز NUPCO التفاعلي
منصة موحدة تستضيف 5 ألعاب تفاعلية للتوعية بالأمن السيبراني.

### تحدي قيم SPL
تجربة ويب ديناميكية تتضمن سبعة تحديات تفاعلية لتعزيز مشاركة الموظفين.
