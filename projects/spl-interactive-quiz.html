Interactive Quiz for SPL's Internal Event

2024-10-07
Web DevelopmentInteractive Experience

Developed an interactive quiz experience for a Saudi Post Logistics (SPL) internal event. The quiz was centered around a physical "vault" with 200 locked boxes, each associated with a unique barcode that triggered a question when scanned.

JavaScriptHTMLCSSJSONGitHub

Features & Functionality

Barcode Scanning

Users could scan a barcode on each box to trigger a unique question.

Question and Answer Logic

The system displayed the question and provided multiple-choice options.

Answer Validation

The user's selected answer was validated in real-time.

Feedback Mechanism

If correct, the user received a congratulatory message and the three-digit code to unlock the corresponding box. If incorrect, the user received a "try again" message (later modified to one attempt per question).
